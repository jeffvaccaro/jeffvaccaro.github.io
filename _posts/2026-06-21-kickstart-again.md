I am rebooting this blog workflow and getting back into writing regularly.

This post is my marker that the pipeline is working again:

1. Write post in `_posts` using the `YYYY-MM-DD-title.md` format.
2. Run `bundle exec jekyll serve` for local preview.
3. Run `bundle exec jekyll build` for a final build check.
4. Push to `main` to trigger GitHub Pages deployment.

More posts coming soon.
